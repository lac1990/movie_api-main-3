Movie API

Movie API provides information to users about movies that consists of synopsis, directors, and bios. Users can interact by signing up, updating their personal information, and liking favorite movies.

Server

Functions

GET
/movies lists all the movies
GET
/movies/{Title} lists all data about a movie by the title
GET
/movies/genre/{Genre} lists all the movie titles with a specific genre
GET
/movies/director/{Director Name} lists data about a director
GET
/users lists all users
POST
/users create a new user
PUT
/users/{UserName} update user name using the UserName
DELETE
/users/{id} remove user from the list
POST
/users/{UserName}/movies/{Title} add a movie to the user's favorite movies list
DELETE
/users/{UserName}/movies/{Title} delete a movie from the user's favorite movies list
